## Prerequisites
```
Python 3.x
```

## How to use
1. Clone repository
   ```
   git clone https://github.com/ksu-is/NBA_Stats.git
   cd NBA_Stats
   ```  
2. Install NBA API
   ```
   pip install nba_api
   ```
3. Run code in VS Code or terminal
   ```
   python NBA_Stats.py
   ```
